<Info>
  **Gereksinimler**:
  - Node.js sürüm 19 veya üzeri
  - `docs.json` dosyası içeren bir dokümantasyon repository'si
</Info>

İşletim sisteminizde Mintlify'ı kurmak ve çalıştırmak için bu adımları izleyin.

<Steps>
<Step title="Mintlify CLI'ı Kurun">

```bash
npm i -g mint
```

Mintlify CLI'ı global olarak yükleyin. Bu araç, dokümantasyonunuzu yerel ortamda önizlemenizi sağlar.

</Step>

<Step title="Yerel Önizleme Başlatın">

`docs.json` dosyasının bulunduğu docs dizinine gidin ve aşağıdaki komutu çalıştırın:

```bash
cd docs
mint dev
```

Dokümantasyonunuzun yerel önizlemesi `http://localhost:3000` adresinde kullanılabilir olacaktır.

</Step>
</Steps>

## Özel Port Kullanımı

Varsayılan olarak Mintlify 3000 portunu kullanır. `--port` bayrağını kullanarak Mintlify'ın çalışacağı portu özelleştirebilirsiniz. Örneğin, Mintlify'ı 3333 portunda çalıştırmak için:

```bash
mint dev --port 3333
```

Halihazırda kullanımda olan bir portta Mintlify'ı çalıştırmaya çalışırsanız, bir sonraki kullanılabilir portu kullanacaktır:

```md
Port 3000 zaten kullanımda. Bunun yerine 3001 deneniyor.
```

## Mintlify Sürümleri

Her CLI sürümünün belirli bir Mintlify sürümüyle ilişkili olduğunu unutmayın. Yerel önizlemeniz production sürümüyle uyuşmuyorsa, lütfen CLI'ı güncelleyin:

```bash
mint update
```

## Link Doğrulama

CLI, dokümantasyonunuzdaki linkleri doğrulamada yardımcı olabilir. Kırık linkleri tespit etmek için aşağıdaki komutu kullanın:

```bash
mint broken-links
```

Bu komut, dokümantasyonunuzdaki tüm linkleri tarar ve çalışmayan linkleri size bildirir.

## Sorun Giderme

<AccordionGroup>
  <Accordion title='Hata: "sharp" modülü darwin-arm64 runtime kullanılarak yüklenemedi'>

    Bu, eski bir node sürümünden kaynaklanıyor olabilir. Şunları deneyin:
    1. Şu anda yüklü olan CLI sürümünü kaldırın: `npm remove -g mint`
    2. Node v19 veya üzerine yükseltin
    3. CLI'ı yeniden yükleyin: `npm i -g mint`
  </Accordion>

  <Accordion title="Sorun: Bilinmeyen bir hatayla karşılaşma">

    Çözüm: Cihazınızın kök dizinine gidin ve `~/.mintlify` klasörünü silin. Ardından `mint dev` komutunu tekrar çalıştırın.
  </Accordion>

  <Accordion title="Port 3000 kullanımda hatası">

    Eğer 3000 portu kullanımdaysa, ya başka bir port belirtin:
    ```bash
    mint dev --port 3333
    ```
    Ya da 3000 portunu kullanan uygulamayı kapatın.
  </Accordion>
</AccordionGroup>

## Fonify.co Projesi için Geliştirme

Fonify.co web sitesi Next.js ile geliştirilmiştir. Web sitesini yerel ortamda çalıştırmak için:

```bash
# Ana proje dizininde
npm install
npm run dev
```

Web sitesi `http://localhost:3000` adresinde çalışacaktır. Dokümantasyon için `docs/` dizininde çalışın.
