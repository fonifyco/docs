## Üç Adımda Başlayın

Fonify.co dokümantasyonunu yerel ortamınızda çalıştırın ve ilk özelleştirmenizi yapın.

### Adım 1: Yerel Ortamınızı Kurun

<AccordionGroup>
  <Accordion icon="copy" title="Dokümantasyonu yerel ortama klonlayın">
    GitHub repository'sini yerel bilgisayarınıza klonlayın:

    ```bash
    git clone https://github.com/DefSoft-AI/fonify.co.git
    cd fonify.co/docs
    ```

    Değişiklikleri yapıp önizlemek için repository'yi yerel ortamınıza klonlamanız gerekmektedir.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Önizleme sunucusunu başlatın">
    1. Mintlify CLI'ı yükleyin: `npm i -g mint`
    2. Docs dizinine gidin: `cd docs`
    3. Geliştirme sunucusunu başlatın: `mint dev`
    4. Tarayıcınızda `http://localhost:3000` adresini açın

    <Tip>Dosyaları düzenledikçe önizleme otomatik olarak güncellenir!</Tip>
  </Accordion>
</AccordionGroup>

### Adım 2: Özelleştirme Yapın

<AccordionGroup>
  <Accordion icon="palette" title="Site adını ve renklerini güncelleyin">
    İlk değişiklik olarak site adını ve renklerini güncelleyelim:

    1. Editörünüzde `docs.json` dosyasını açın
    2. `"name"` alanını proje adınızla değiştirin
    3. `"colors"` bölümünü markanıza uygun şekilde güncelleyin
    4. Kaydedin ve değişiklikleri `http://localhost:3000` adresinde görün

    <Tip>Ana rengi değiştirerek anında farkı görebilirsiniz!</Tip>
  </Accordion>
  <Accordion icon="map" title="Navigasyon yapısını düzenleyin">
    Dokümantasyonunuzun navigasyon yapısını özelleştirin:

    1. `docs.json` dosyasında `navigation` bölümünü bulun
    2. Sayfa gruplarını ve sayfaları düzenleyin
    3. Gerektiğinde yeni sayfalar ekleyin veya mevcut sayfaları kaldırın

    <Tip>Kullanıcılarınızın ihtiyaç duydukları bilgileri kolayca bulmalarını sağlayın.</Tip>
  </Accordion>
</AccordionGroup>

### Adım 3: Yayınlayın

<Accordion icon="rocket" title="Değişikliklerinizi yayınlayın">
  1. Değişikliklerinizi commit edin
  2. GitHub'a push yapın
  3. Dokümantasyonunuz otomatik olarak güncellenecektir!

  ```bash
  git add .
  git commit -m "Dokümantasyon güncellendi"
  git push
  ```
</Accordion>
